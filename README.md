# Marathon Heart Rate Tattoo Generator

## Overview
This repository contains an interactive web tool and Python data pipeline designed to visualize heart rate data from a 5-hour, 11-minute marathon run. The primary goal of this project is to generate a customizable, scalable vector graphic (SVG) of the heart rate profile to serve as a stencil for a continuous line tattoo running vertically up the right side of the body, from the ankle to the wrist. 

## Features
* **Data Processing Pipeline:** A Python script (`etl.py`) that filters raw, full-day Fitbit heart rate CSV data to isolate the specific timeframe of the marathon and generate static reference plots.
* **Interactive Design Tool:** A single-file, D3.js-powered web interface that allows for real-time manipulation of the graph's visual parameters.
* **Vertical Orientation:** A custom layout mapping the data from the starting line (bottom) to the finish line (top) to physically align with the intended placement of the tattoo.
* **SVG Export:** Direct-to-browser SVG downloading provides a clean, lossless vector stencil ready for a tattoo artist to use.

## Repository Contents
* `etl.py`: The Python script used to extract, filter, and plot the raw data.
* `heart_rate_2017-10-08.csv`: The original, raw full-day heart rate data.
* `marathon_heart_rate_data.csv`: The filtered dataset containing only the marathon's time window (generated by `etl.py`).
* `marathon_heart_rate_vertical_interactive.html`: The complete, self-contained interactive D3.js designer.
* `README.md`: Project documentation and usage instructions.

## Tool Parameters (Interactive Web View)
The interactive web tool includes the following tunable parameters to help shape the perfect stencil:
* **Total Length (Vertical):** Adjusts the overall height of the SVG to simulate the physical distance along the leg, torso, and arm.
* **Graph Width (Amplitude):** Controls the horizontal spread and intensity of the heart rate peaks.
* **Line Thickness:** Simulates different tattoo needle groupings and line weights.
* **Smoothing (Moving Average):** Reduces the jaggedness of raw biometric data to create cleaner, flowing aesthetic curves.
* **Clamp Low BPM (Cutoff):** Establishes a baseline cutoff to remove lower heart rate noise and tighten the visual structure of the line.

## Usage
### Running the Python Pipeline
1. Ensure you have Python installed with the required libraries (`pandas`, `matplotlib`).
2. Place your raw full-day heart rate data in the directory as `heart_rate_2017-10-08.csv`.
3. Run `python etl.py` to generate the filtered CSV dataset and static reference PNGs.

### Using the Interactive Designer
1. Open `marathon_heart_rate_vertical_interactive.html` in any modern web browser (Chrome, Safari, Firefox, etc.).
2. Adjust the sliders in the control panel to achieve the desired aspect ratio, line smoothness, and amplitude.
3. Click **"Download SVG for Tattoo Artist"** to export the final vector design to your machine.
